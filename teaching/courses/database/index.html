数据库原理

课程信息

  • 课程编号：CS150
  • 学期：2024年春季
  • 上课时间：周三 6-7节，周五 1-2节
  • 上课地点：第三教学楼201
  • 办公时间：周三 2-4pm 或预约
  • 办公地点：同济经管大厦A楼1209

    • 课程简介

  • 数据库原理课程旨在介绍数据库系统的基本概念、理论、设计和应用。课程涵盖关系数据库的结构、操作和优化方法，帮助学生掌握数据库管理系统（DBMS）的原理，理解数据存储、检索和管理的方法。本课程不仅提供理论基础，还包括实践操作，学生将学习使用 SQL 进行数据库查询和管理，体验数据库的实际应用。

    • 课程目标

    • 1.理解数据库系统的基本概念
    • 2.掌握关系数据库设计方法
    • 3.熟练使用 SQL 进行数据库操作
    • 4.理解数据库事务管理和并发控制
    • 5.了解数据库的性能优化方法

    预修要求

    • 1.C语言程序设计基础
    • 2.基础的数学知识（离散数学、概率统计）

    考核方式
